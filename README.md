# Customer Churn Intelligence Dashboard

End-to-end customer churn analytics solution using SQL, Python, Machine Learning, and Power BI to identify churn drivers, high-risk customer segments, and revenue exposure.

---

## Business Problem

Customer churn directly impacts revenue and long-term business growth.  
This project analyzes customer behavior patterns to identify churn risks, high-risk customer groups, and revenue exposure.

---

## Tech Stack

- SQL
- Python
- Pandas
- Scikit-learn
- Power BI
- Machine Learning

---

## Machine Learning Models

- Logistic Regression
- Decision Tree

---

## Key Insights

- Month-to-month contracts showed the highest churn risk
- Early-tenure customers were more likely to churn
- Identified approximately €4.4M revenue at risk
- Electronic check users demonstrated elevated churn behavior

---

## Dashboard Features

- KPI monitoring
- Churn segmentation
- Revenue risk analysis
- Customer risk visualization
- Interactive filtering and drill-down analysis
